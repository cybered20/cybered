Hacking Metasploitable

What is this learning path? What is metasploitable?

Metasploitable is a virtual machine (VM) sporting Linux as an OS. It's loaded with many vulnerabilities which could be found out in the wild. By experimenting with this VM one can learn about basic vulnerabilities all while getting to play with the metasploit framework, a tool used by many penetration testers / ethical hackers.
